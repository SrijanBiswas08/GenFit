Cross-Body Cable Extension

Description

This isolation exercise targets the triceps by focusing on the long head through a unique cross-body motion using a single cable handle.

Posture Guide

- Set a cable pulley at shoulder height
- Stand sideways to the machine and grab the handle with the opposite hand
- Pull the cable across your body while keeping your elbow stationary
- Squeeze the triceps at the end and return slowly to the start
